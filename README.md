# Pendahuluan

Project ini dibuat sebagai latihan aku untuk mempelajari framework Svelte

# Menggunakan Code

Untuk menggunakan kode setidaknya kamu harus punya npm karena dia merupakan bagian dari node modules

1. Clone repository atau download dalam bentuk zip project ini
2. Buka terminal dan cd kedalam folder project anda
3. Jalankan perintah npm install untuk menginstall package-package
```sh
npm install  
```
4. Jalankan server
```sh
npm run dev -- --open
```
5. Jika kamu ingin menghosting kode dan menjalankan nya secara static maka perlu di build (Note:konfigurasi project ini diseting untuk hasil static)
```sh
npm run build
```
6. Menampilkan hasil build
```sh
npm run preview -- --open
```
sekarang kode yg jalan bukan merupakan hasil dari build stattic yang bisa digunakan untuk hosting
